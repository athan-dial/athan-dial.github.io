A growing collection of Claude Code plugins — extracted from real workflows, packaged for reuse.

Each plugin has its own subsite with docs, a quickstart guide, and a commands reference. Click through to explore.

## Plugins

### [orc](/skills/orc/)

Multi-agent orchestration for Claude Code. Break complex work into parallel sub-tasks dispatched to Cursor, Codex, or Claude subagents with wave-based execution, persistent state, and clean session handoff.

`claude plugin install athan-dial/skills:orc`

### [folio](/skills/folio/)

Manuscript orchestration pipeline. Converts research ideas and messy materials into validated, exportable manuscripts through artifact-driven stages, quality gates, and human checkpoints. Supports white paper, research paper, and hybrid modes.

`claude plugin install athan-dial/skills:folio`
